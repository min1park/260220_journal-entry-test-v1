주요 기능

  • 투명 배경으로 작업 중에도 시간 확인
  • 드래그로 위치 이동, 마우스 휠로 크기 조절
  • 빠른 시작 (1분~60분 프리셋)
  • 폰트, 색상, 외곽선, 투명도 커스터마이징
  • 잔여 시간에 따라 색상 자동 변경 (30초↓ 노랑, 10초↓ 빨강)
  • 시스템 트레이 지원
