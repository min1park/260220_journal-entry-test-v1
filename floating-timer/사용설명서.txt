플로팅 타이머 v1.0 사용 설명서

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 프로그램 소개
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

플로팅 타이머는 컴퓨터 화면 위에 투명하게 떠 있는 카운트다운 타이머입니다. 
작업 중에도 항상 화면 앞에 표시되어 남은 시간을 한눈에 확인할 수 있습니다. 
배경이 투명하여 숫자만 보이며, 위치와 크기를 자유롭게 조절할 수 있습니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. 설치 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"플로팅타이머_설치.exe" 파일을 더블클릭하면 설치 마법사가 시작됩니다. 화면의 
안내에 따라 "다음"을 클릭하고 "설치"를 누르면 설치가 완료됩니다. 설치가 끝나면 
바탕화면에 "플로팅 타이머" 아이콘이 자동으로 생성되며, Windows 시작 메뉴에도 
등록됩니다.

처음 실행할 때 "Windows의 PC 보호"라는 파란색 보안 경고 화면이 나타날 수 
있습니다. 이는 개인 개발 프로그램에 공통적으로 나타나는 정상적인 안내이며, 
"추가 정보"를 클릭한 뒤 "실행"을 누르면 정상적으로 실행됩니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3. 프로그램 실행
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

바탕화면의 "플로팅 타이머" 아이콘을 더블클릭하면 프로그램이 실행됩니다. 
화면 중앙 상단에 흰색 숫자 "00:00"이 나타나며, 배경은 투명하여 숫자만 
표시됩니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4. 기본 조작법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4.1 위치 이동

숫자를 마우스 왼쪽 버튼으로 클릭한 채 원하는 방향으로 드래그하면 화면 내 
어디든 이동시킬 수 있습니다.


4.2 크기 조절

크기를 조절하는 방법은 두 가지입니다. 

첫 번째 방법은 숫자 위에서 마우스 휠을 위로 굴리면 커지고 아래로 굴리면 
작아집니다. 

두 번째 방법은 마우스를 타이머 위에 올리면 하단에 작은 점들이 나타나는데, 
이 부분을 마우스로 잡고 아래로 드래그하면 커지고 위로 드래그하면 작아집니다. 

최소 16pt부터 최대 300pt까지 조절 가능합니다.


4.3 메뉴 열기

숫자를 마우스 오른쪽 버튼으로 클릭하면 설정 메뉴가 열립니다. 모든 기능은 
이 우클릭 메뉴를 통해 사용할 수 있습니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. 타이머 사용법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

5.1 빠른 시작

우클릭 메뉴에서 "빠른 시작"에 마우스를 올리면 미리 설정된 시간 목록이 
나타납니다. 1분, 3분, 5분, 10분, 15분, 25분, 30분, 45분, 60분 중 원하는 
시간을 클릭하면 즉시 카운트다운이 시작됩니다.


5.2 직접 입력

우클릭 메뉴에서 "직접 입력"을 선택하면 시간 입력 창이 나타납니다. 시, 분, 
초를 원하는 만큼 입력하고 ▶ 버튼을 누르거나 키보드의 Enter 키를 누르면 
카운트다운이 시작됩니다.


5.3 일시정지와 계속

타이머가 작동 중일 때 우클릭 메뉴에서 "일시정지"를 선택하면 타이머가 
멈춥니다. 다시 "계속"을 선택하면 멈춘 시점부터 이어서 카운트다운이 
진행됩니다.


5.4 리셋

우클릭 메뉴에서 "리셋"을 선택하면 타이머가 00:00으로 초기화됩니다.


5.5 시간 표시 형식

설정한 시간이 1시간 미만인 경우 "MM:SS" 형식으로 표시됩니다. 
예를 들어 25분은 "25:00"으로 나타납니다. 

설정한 시간이 1시간 이상인 경우 "HH:MM:SS" 형식으로 표시됩니다. 
예를 들어 1시간 30분은 "01:30:00"으로 나타납니다.


5.6 타이머 완료 알림

카운트다운이 끝나면 숫자가 빨간색으로 깜빡이며 알림 소리가 울립니다. 

또한 잔여 시간에 따라 글자 색상이 자동으로 변경되어 시간이 얼마나 남았는지 
직관적으로 알 수 있습니다. 잔여 시간이 30초 이하가 되면 노란색으로 바뀌고, 
10초 이하가 되면 빨간색으로 바뀝니다.

6. 꾸미기 기능
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

6.1 폰트 변경

우클릭 메뉴에서 "폰트 변경"을 선택하면 8가지 폰트 중 원하는 것을 선택할 수 
있습니다. Arial Black, Impact, Consolas, Segoe UI Black, Verdana, Tahoma, 
Franklin Gothic Heavy, Bahnschrift를 지원합니다. 기본 폰트는 Arial Black입니다.


6.2 외곽선 두께

우클릭 메뉴에서 "외곽선 두께"를 선택하면 글자 테두리의 두께를 조절할 수 
있습니다. 없음, 얇게, 보통, 굵게, 매우 굵게 중 선택 가능합니다. 외곽선이 
있으면 어떤 배경 위에서도 글자가 선명하게 보입니다. 글자 크기를 변경하면 
외곽선 두께도 비례하여 자동 조절됩니다.


6.3 글자 색상 변경

우클릭 메뉴에서 "글자 색상"을 선택하면 흰색, 초록, 파랑, 노랑, 주황, 빨강, 
보라, 핑크 중 원하는 색을 선택할 수 있습니다. 기본 색상은 흰색입니다.


6.4 투명도 조절

우클릭 메뉴에서 "투명도"를 선택하면 타이머 전체의 투명도를 조절할 수 
있습니다. 100%부터 30%까지 설정 가능하며, 숫자를 낮출수록 타이머가 연하게 
보입니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
7. 시스템 트레이
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

7.1 트레이로 숨기기

우클릭 메뉴에서 "트레이로 숨기기"를 선택하면 화면에서 타이머가 사라지고, 
화면 오른쪽 하단 시계 옆 알림 영역에 시계 모양 아이콘이 나타납니다. 타이머가 
작동 중이었다면 숨겨진 상태에서도 카운트다운은 계속 진행됩니다.


7.2 트레이에서 다시 열기

알림 영역의 시계 모양 아이콘을 클릭하면 "타이머 열기"와 "종료" 메뉴가 
나타납니다. "타이머 열기"를 선택하면 타이머가 다시 화면에 표시됩니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
8. 프로그램 종료
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

우클릭 메뉴에서 "완전 종료"를 선택하면 프로그램이 종료됩니다. 다시 사용하려면 
바탕화면의 아이콘을 더블클릭하면 됩니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
9. 프로그램 제거
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Windows 설정 → 앱 → "플로팅 타이머"를 찾아 "제거"를 클릭하면 프로그램이 
깔끔하게 삭제됩니다. 시작 메뉴의 "플로팅 타이머" 폴더에서 "플로팅 타이머 
제거"를 선택해도 동일하게 제거할 수 있습니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
